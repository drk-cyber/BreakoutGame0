# 🧱 Java 打砖块游戏 (Breakout)

> 一个经典打砖块游戏，支持多关卡、道具系统和本地高分榜。  

---

## 📖 项目简介

打砖块是一款经典的街机游戏：玩家控制屏幕底部的挡板反弹小球，击碎屏幕上方的所有砖块。  
本项目在基础玩法之上增加了：

- 动态生成的无限关卡（每关砖块布局递增）
- 多种砖块类型（普通、坚固、爆炸、再生）
- 随机掉落的道具（加宽挡板、额外生命、加速球）
- 本地高分持久化（保存前 10 名分数）
- 暂停/继续、游戏结束重启等完整交互

---

## 🎮 游戏玩法

- **目标**：用球击碎所有砖块，每关清空后自动进入下一关。
- **生命**：初始 3 条命，球落底扣 1 条，生命归零则游戏结束。
- **分数**：每击碎一块砖获得分数（普通 100 分，坚固 300 分），关卡升级额外奖励。
- **砖块**:  
  - **砖块数量**:	第 1 关 56 块 → 第 12 关 168 块  
  - **坚固砖块**:	需要 3 次击中  
  - **再生砖块**:	打碎后 5 秒原地复活  
  - **爆炸砖块**:	摧毁时波及周围  
- **道具**：砖块破碎时 5% 概率掉落道具，接到后获得增益：
  - 🟢 **加宽挡板**：挡板宽度增加 40 像素（上限 400px）
  - 🩷 **额外生命**：生命 +1
  - 🟠 **加速球**：立即获得 50 分（演示占位）

## 🎛️ 操作说明

| 按键 | 功能 |
|------|------|
| `←`  | 挡板左移 |
| `→`  | 挡板右移 |
| `Space` | 发射球 / 暂停/继续 |
| `R` | 重新开始（游戏结束后可重置） |

##  📦项目源码结构
Breakout  
├── 📄 BreakoutMain.java          # 🚀 程序入口（SwingUtilities 启动）  
├── 📄 GameWindow.java            # 🪟 JFrame 窗口封装（含 BufferStrategy 初始化）  
├── 📄 GamePanel.java             # 🎮 核心：游戏循环、线程、渲染、输入、碰撞调度（约 400 行）  
├── 📄 GameConstants.java         # ⚙️ 全局常量（窗口尺寸、FPS、标题）  
│  
├── 📄 Entity.java                # 🧬 抽象基类（位置、边界、update/render 契约）  
├── 📄 Ball.java                  # ⚪ 球：移动、发射、碰撞响应（挡板/砖块/边界）  
├── 📄 Paddle.java                # 🟧 挡板：键盘移动、边界限制  
│  
├── 📄 Brick.java                 # 🧱 抽象砖块 + 普通砖 (Normal) + 坚固砖 (Strong)  
├── 📄 AdvancedBricks.java        # 💥 扩展砖块：爆炸砖 (Exploding) + 再生砖 (Regenerating)  
├── 📄 PowerUp.java               # ⭐ 道具基类 + 具体实现（加宽/加命/加速球）  
│  
├── 📄 LevelManager.java          # 🗺️ 关卡生成（行列算法）、进度检测、下一关推进  
├── 📄 ScoreManager.java          # 📊 分数累加、生命管理、关卡奖励  
├── 📄 HighScoreManager.java      # 🏆 高分文件读写、排序、持久化（容量 10）  
├── 📄 Menu.java                  # 📋 菜单绘制（主菜单/暂停/结束覆盖层） 
│  
└── 📄 highscores.txt             # 💾 高分数据文件（自动生成，每行一个整数）

##  🔧 参数调优（魔改指南）
| 参数 | 位置 | 默认 | 魔改建议 |  
|------|------|------|------|
| 球速 | Ball.speed | 370 | 调高 = 地狱模式 |  
|挡板速度	|Paddle.speed|	400	|调高 = 闪现挡板|  
|砖块最大行/列	|LevelManager|	12/14	|调高 = 砖块海|  
|道具掉落率	|GamePanel|	5%	|调高 = 道具雨|  
|初始生命	|ScoreManager.lives|	3|	调高 = 苟命王|
